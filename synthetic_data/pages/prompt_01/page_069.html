First selfbow - osage orange - took set after 30 arrows, bracing height changed
Selfbow • Posted by BowBuilder_Connor • June 2001

I built my first selfbow from an osage orange stave - 62 inches long, roughly 50 lb draw at 28 inches. Tillered it carefully over two weeks. After the first shooting session of about 30 arrows, the bow "followed the string" - the limbs bent more toward the string side at rest than when I started. Bracing height dropped by about 1/2 inch. The bow still shoots but feels different. Is this normal for a new selfbow? Is the set permanent? Did I damage the bow?

Connor, string follow (set) in a new osage selfbow is completely normal - not damage. Here's the explanation:

Wood is viscoelastic - it creeps slightly under load over time. Every new selfbow, especially one fresh to string, takes some initial set in the first shooting sessions. Osage orange is one of the best selfbow woods in the world precisely because it takes LESS set than most alternatives - but no wood is immune to initial set.

A 1/2-inch drop in bracing height after 30 arrows is modest and acceptable. The bow is settling into its working position. Some bowers call this "working in" the bow. Most of the initial set will be in the first few sessions; after 200-300 arrows the set typically stabilizes.

Adding to Silas: osage orange has exceptional compression and tension properties which is why it was prized by Native American bowyers. It does take set in fresh staves, and more so if the stave wasn't thoroughly dried (was yours air-dried at least 1-2 years, or kiln-dried?). Green or partially dried osage takes significantly more set than fully seasoned material. If your stave was well-seasoned, the set will stabilize. If it was fresh-cut, you may see more movement over time.

MasterFletch_Silas

Adjust the string to restore your original bracing height - you can add a few twists to the string to shorten it. Shoot at your intended bracing height, not the set-down height. The set height is where the limbs want to rest, but you brace the bow to a specific height for performance.

To minimize future set: store the bow unstrung when not in use. Leaving a selfbow strung permanently adds hours of load and causes progressive string follow. Unstring after every session. This is traditional practice and significantly reduces long-term set.

Regarding the "different feel" after set: many archers report that a broken-in selfbow with some set feels smoother and more forgiving to shoot than a perfectly stiff new bow. The limbs have taken their working shape and settled. Some amount of set is expected and aesthetically part of a traditional selfbow. A bow with absolutely zero set after use would be unusual.

Tillering a 50 lb osage bow from a stave as a first build is ambitious. That's real bowyer work, not just stringing a manufactured bow. The fact that it shoots and the set is modest suggests your tillering was actually quite good. Most first selfbows either chrysal (fracture) or take dramatic set. A 1/2" set after 30 arrows is a successful bow.
