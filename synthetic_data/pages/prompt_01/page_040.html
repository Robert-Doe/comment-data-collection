BEFORE DESOLDERING THE SID, CHECK THESE THINGS FIRST: 1) AUDIO FILTER CAPS (C38-C42 ON MOST BREADBIN BOARDS) - THEY GO DRY AND BLOCK AUDIO 2) THE VOLUME REGISTER - OPEN MONITOR, TYPE POKE 54296,15 AND SEE IF ANY SOUND APPEARS 3) THE AUDIO OUTPUT TRANSISTOR (Q9) - OFTEN FAILS AND IS MUCH EASIER TO REPLACE THAN THE SID IF POKE 54296,15 PRODUCES NOTHING, THE SID IS SUSPECT. IF IT PRODUCES NOISE BUT NO MUSIC, THE PROBLEM IS DOWNSTREAM (CAPS OR TRANSISTOR).

ROB IS RIGHT - CHECK Q9 FIRST. A FAILED Q9 TRANSISTOR IS A KNOWN FAILURE MODE ON BREADBIN BOARDS AND IS A CHEAP COMMON BC547 TRANSISTOR. I'VE FIXED FOUR SILENT C64S THIS WAY WITHOUT TOUCHING THE SID. THE SID ITSELF RARELY FAILS COMPLETELY - PARTIAL FAILURE (ONLY SOME VOICES DEAD) IS MORE COMMON.

SILENCE FROM POKE IS BAD - STRONGLY SUGGESTS SID IS NOT GENERATING AUDIO. BUT BEFORE DESOLDERING: CAREFULLY MEASURE VOLTAGE ON SID PIN 28 (VCC) - SHOULD BE 12V. IF YOU'RE GETTING 12V ON PIN 28 AND STILL SILENCE FROM THE POKE COMMAND, THE SID IS ALMOST CERTAINLY FAILED. SOURCE A REPLACEMENT 6581 OR CONSIDER THE MODERN FPGASID OR SID2SID REPLACEMENT. THE CAPS: REPLACE WITH SAME VALUE ELECTROLYTICS (47UF/25V IS TYPICAL). LOW ESR MODERN CAPS ARE FINE.

THE 6581 AND 8580 ARE DIFFERENT CHIPS WITH DIFFERENT FILTER CHARACTERISTICS - THE 8580 HAS EXTERNAL FILTER CAPS ON THE BOARD WHILE THE 6581 HAS THEM INTERNALLY. MAKE SURE YOU SOURCE THE CORRECT REVISION FOR YOUR BOARD (ASSY NUMBER ON PCB TELLS YOU WHICH SID REVISION IT EXPECTS). MIXING THEM CAN WORK BUT FILTER RESPONSE WILL DIFFER.
