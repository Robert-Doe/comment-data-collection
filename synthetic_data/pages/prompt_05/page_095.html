ParkBench

Park redesign comments settle into a long chain

A local community page uses a late-2000s comment UI with enough structure for replies, votes, and a hidden preview.

The page is built to show how jQuery-era teams wrapped behavior around meaningful static HTML.

It keeps the comment block parseable without JavaScript while still exposing ids, data attributes, and hook classes for enhancement.
